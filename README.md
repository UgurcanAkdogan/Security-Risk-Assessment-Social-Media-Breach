# Security-Risk-Assessment-Social-Media-Breach
A comprehensive security risk assessment for a social media organization following a major data breach. This project identifies vulnerabilities and recommends network hardening strategies like MFA, strict password policies, and firewall configurations.
Project Overview

This project focuses on analyzing a significant data breach at a social media organization. As a security analyst, I conducted a risk assessment to identify key vulnerabilities that led to the compromise of customer information and recommended a series of network hardening tasks to prevent future incidents.
Identified Vulnerabilities

Through a detailed audit, four critical security gaps were discovered:

    Poor Password Hygiene: Employee password sharing.

    Weak Credentials: Default admin passwords for database access.

    Lack of Network Filtering: Missing firewall rules for traffic control.

    Single-Factor Authentication: Absence of Multifactor Authentication (MFA).

Recommended Hardening Strategies

To secure the organization, I recommended the following implementations:

    Multifactor Authentication (MFA): Adding a second layer of security to prevent unauthorized access even if passwords are stolen.

    Strict Password Policy: Mandatory complex passwords and immediate removal of all default credentials.

    Firewall ACL Configurations: Implementing Access Control Lists to filter and block suspicious network traffic.
